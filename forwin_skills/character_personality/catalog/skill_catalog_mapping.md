# Skill Catalog Mapping v2

## Traits

| Skill | Chinese Name | Core Mechanism | Required Focus |
|---|---|---|---|
| `trait-cautious-strategist` | 审慎策士 | 用预案、信息和延迟承诺换安全 | 风险评估、错失窗口、如何冒险 |
| `trait-impulsive-hothead` | 冲动火爆者 | 用快速行动和高情绪强度破局 | 被挑衅反应、事后补救、关系损伤 |
| `trait-suspicious-survivor` | 多疑幸存者 | 用信息优势、试探和退路维持安全 | 信任更新、保密、过度怀疑 |
| `trait-loyal-protector` | 忠诚守护者 | 用承诺和保护建立归属 | 护短、边界、牺牲与成长 |
| `trait-ambitious-climber` | 野心攀登者 | 用成就和上升确认价值 | 地位敏感、功利化、健康野心 |
| `trait-manipulative-charmer` | 操控魅力者 | 用读场、吸引、共情和包装换控制 | 情绪利用、暧昧边界、真诚裂缝 |
| `trait-principled-judge` | 原则审判者 | 用规则、秩序和正当性抵御混乱 | 苛责、温度、例外处理 |
| `trait-pragmatic-opportunist` | 务实投机者 | 用机会、交换和灵活策略最大化收益 | 短视、信用、底线 |
| `trait-stoic-repressor` | 隐忍压抑者 | 用功能稳定压住情绪和伤口 | 身体动作、延迟爆雷、求助能力 |
| `trait-volatile-explosive` | 易爆反应者 | 用爆发强度挡羞耻、无力或恐惧 | 伤害成本、修复责任、触发阈值 |
| `trait-quiet-observer` | 沉静观察者 | 先观察、记忆、理解，再进入关系或行动 | 存在感低、细节感、迟到的决定 |
| `trait-social-butterfly` | 社交蝴蝶 | 用连接、气氛和流动性换机会与能量 | 浅层关系、承诺过量、真心时刻 |
| `trait-analytical-skeptic` | 分析型怀疑者 | 用证据、逻辑和漏洞识别防犯错 | 共情不足、过度校验、被说服条件 |
| `trait-curious-explorer` | 好奇探索者 | 用探索未知和试错获取生命力 | 越界、厌常规、责任收束 |
| `trait-duty-bound-commander` | 责任型指挥者 | 用责权、纪律和任务完成维持秩序 | 控制、命令、责任过载 |
| `trait-shame-driven-overachiever` | 羞耻驱动型过度成就者 | 用优绩遮住“不够好”的羞耻 | 完美主义、KPI 化关系、健康自尊 |

## Stress Modes

| Skill | Chinese Name | Trigger Core | Required Focus |
|---|---|---|---|
| `stress-paranoid-controller` | 偏执控制 | 信息缺口、背叛疑云、失控感 | 关系成本、恢复控制权的健康方式 |
| `stress-violent-outburst` | 暴力爆发 | 羞辱、压迫、累积挫败 | 伤害边界、降温、事后责任 |
| `stress-emotional-withdrawal` | 情绪抽离 | 冲突过载、亲密威胁、无力感 | 冷处理成本、恢复窗口 |
| `stress-self-sacrifice` | 牺牲透支 | 他人痛苦、自我价值危机 | 边界、透支、被需要成瘾 |
| `stress-dissociated-analyst` | 解离分析 | 高压与高羞耻并存 | 像机器一样处理，如何回身体 |
| `stress-pleasing-collapse` | 讨好崩塌 | 拒绝线索、强权压力、关系不稳 | 失语、迎合、边界训练 |
