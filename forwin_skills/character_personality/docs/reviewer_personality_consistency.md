# Reviewer Personality Consistency

Reviewer checks should use `active_personality_context`, not the whole skill library.

## Checklist

1. Does the character's decision follow the active dominant skill?
2. If not, is there a canon, relationship, stress, or growth reason?
3. Does dialogue match the expression layer?
4. Does body language match the expression layer, or does the scene rely only on dialogue?
5. Does the stress response match the active pressure level?
6. Does the character behave differently toward different relationship targets?
7. Did the scene turn a tendency into an absolute rule?
8. Did the prose flatten the character into repeated catchphrases?
9. Did a reference model label override actual behavior rules?
10. Did the skill invent new canon facts?
11. Did a social mask become the character's entire self?
12. Did a stress mode stay active without a trigger?
13. Did growth erase the personality mechanism instead of making it healthier?
