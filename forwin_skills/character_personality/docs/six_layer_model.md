# Six Layer Personality Model v2

Every reusable personality skill is organized around six layers.

```text
Layer 1: Baseline trait axes
Layer 2: Core drive
Layer 3: Decision mechanics
Layer 4: Relationship pattern
Layer 5: Expression
Layer 6: Stress and arc
```

## Layer 1: Baseline Trait Axes

Trait axes are stable tendencies. They do not decide outcomes by themselves.

Required baseline dimensions for traits:

```yaml
trait_axes:
  openness: TBD
  conscientiousness: TBD
  extraversion: TBD
  agreeableness: TBD
  emotional_volatility: TBD
  honesty_humility: TBD
  dominance: TBD
  risk_tolerance: TBD
  trust_baseline: TBD
  intimacy_need: TBD
  rule_respect: TBD
  status_sensitivity: TBD
```

Always translate trait axes into writable behavior: what the character notices first, avoids, delays, tests, hides, or protects.

## Layer 2: Core Drive

Core drive explains why the behavior exists.

```yaml
core_drive:
  external_want: TBD
  internal_need: TBD
  core_fear: TBD
  core_shame: TBD
  core_lie: TBD
  compensation_strategy: TBD
```

Do not write only "proud" or "cold." Write what pride protects, what breaks it, what cost it creates, and how growth changes it.

## Layer 3: Decision Mechanics

Decision mechanics define:

```text
trigger -> interpretation -> response -> cost -> possible growth response
```

Personality controls tendencies, not conclusions. A suspicious character can trust someone if canon and relationship state support it, but the trust should still show verification, fallback plans, or bounded disclosure.

## Layer 4: Relationship Pattern

Every skill should distinguish at least:

- stranger
- ally
- loved_one
- enemy

Important skills should also cover:

- superior
- subordinate
- rival
- betrayer

## Layer 5: Expression

Expression controls how the character is heard and seen. It must include:

- dialogue behavior
- body language
- affection style
- anger style
- lie style
- silence style
- humor style

Do not rely on dialogue alone. Stable fictional characters are often recognized through movement, posture, touch response, silence, timing, and spatial choices.

## Layer 6: Stress and Arc

Stress and arc define:

- stress triggers
- mild pressure
- medium pressure
- extreme pressure
- breakdown signals
- recovery conditions
- healthy growth
- negative arc
- relapse trigger

Growth is not personality replacement. It is the same mechanism expressed in a healthier form.
