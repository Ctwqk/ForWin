# Relationship Pattern Template

Relationship patterns define intimacy and boundary mechanics. They do not store live relationship scores.

Required extension:

```yaml
relationship_pattern_specific:
  attachment_logic: TBD
  trust_logic: TBD
  boundary_logic: TBD
  dependency_logic: TBD
  jealousy_logic: TBD
  conflict_logic: TBD
  reconciliation_logic: TBD
```
