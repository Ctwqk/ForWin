# Archetype Template

Archetypes define narrative function only. They cannot decide dialogue or day-to-day behavior by themselves.

Required extension:

```yaml
archetype_specific:
  narrative_function: TBD
  story_promise: TBD
  default_conflict_role: TBD
  positive_function: TBD
  negative_function: TBD
  relation_to_protagonist: TBD
```
